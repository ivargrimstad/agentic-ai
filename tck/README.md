# Jakarta Agentic Artificial Intelligence
The Jakarta Agentic AI project provides a set of vendor-neutral APIs that make it easy, consistent, and reliable to build, deploy, and run AI agents on Jakarta EE runtimes.

# Jakarta Agentic AI TCK

This module contains the Technology Compatibility Kit (TCK) for Jakarta Agentic AI. The TCK verifies compatibility of implementations with the specification.

## Building

To build the TCK module:

```
mvn clean package
```
